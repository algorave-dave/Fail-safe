 I left notes throughout the file to help with understanding how I wrote everything -- like noting where certain elements are and which code sections you can mostly ignore. Everything is obviously editable, so go crazy! If you encounter any technical difficulties when executing this file just shoot me a message on my [Discord](https://discord.gg/ZU6SUs6Mem)  and I'll help you.
